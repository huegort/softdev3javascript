Assesment 01

Clientside CRUD

Title:
A Film webpage (CRUD)
Description:

Write a webpage that allows the user to perform CRUD operations on data of films

A film will have

  • id
  • name (String)
  • genre (picked from a list)
  • date published (date)

The website will allow the user to:

  1. View all the Films in a table
  2. Create a new Film
  3. Delete a Film from the table
  4. Update a Film in the table

The web page does not need to store the data on a server. The page should look nice and be intuitive to use
